# Multi-Source Feedback Intelligence System

• Built an AI-powered Feedback Intelligence System that analyzes user reviews using NLP techniques, performs sentiment classification, extracts key issues, and visualizes trends via an interactive Streamlit dashboard.

• Implemented automated PDF report generation with real-time insights, enabling structured analysis of customer feedback from Google Play Store data.

# Features
- Fetches real-time app reviews from Google Play Store
- Performs sentiment analysis (Positive/Negative/Neutral)
- Extracts common issues using NLP
- Visualizes trends over time
- Generates downloadable PDF reports

# Tech Stack
- Python
- Streamlit
- Scikit-learn
- NLTK
- ReportLab

# Use Case
Helps developers and product teams understand user feedback and improve applications based on data-driven insights.
